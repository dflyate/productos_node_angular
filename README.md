### Descargar el proyecto


### PARA EL FRONTEND

- Ingresar a la carpeta `producto_front/productos`
- Abrir una consola de comandos y ejecutar:


```bash

```

- Abrir la ruta generada desde un navegador: `http://localhost:4200/productos`
